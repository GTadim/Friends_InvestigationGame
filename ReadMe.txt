Como acessar o jogo localmente
1. Acessar a pasta html
2. Selecionar Index.html > Abrir com (Internet Edge ou Google Chrome)

Regras do Jogo
1. Você não pode fazer a mesma pergunta para o suspeito duas vezes
2. São permitidas apenas 4 perguntas para cada suspeito
3. O Botão "Prender Suspeito" só será habilitado uma vez que você esgotar as perguntas para cada um dos 5 suspeitos.
4. Uma vez clicado no botão mencionado, você deve escolher um dos personagens para prender.
